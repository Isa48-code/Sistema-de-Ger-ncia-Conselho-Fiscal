# 💰 Sistema de Controle Financeiro - Conselho Fiscal

Este projeto é um sistema simples de controle financeiro desenvolvido com **HTML, CSS e JavaScript**, com foco em atender as necessidades de um **Conselho Fiscal de comunidade (quilombo)**.

## 📌 Objetivo

Permitir que conselheiros fiscais possam:

* Registrar entradas (receitas)
* Registrar saídas (despesas)
* Visualizar saldo atualizado
* Organizar lançamentos com data
* Excluir registros
* Simular uma planilha financeira simples

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* LocalStorage (armazenamento no navegador)

---

## 🖥️ Funcionalidades

### ✔️ Cadastro de lançamentos

* Data
* Descrição
* Valor
* Tipo (Entrada ou Saída)

### ✔️ Controle de saldo

* Soma automática de entradas
* Subtração automática de saídas

### ✔️ Tabela de registros

* Exibição em formato de planilha
* Valores formatados (R$)

### ✔️ Exclusão de registros

* Remoção individual de lançamentos

### ✔️ Armazenamento local

* Dados salvos no navegador (LocalStorage)
